# GDF
> GDF : Graph Data Digest Document Format

## Manual installation

Requirements:

- [Node 8+](https://nodejs.org/en/download/package-manager/)

Node dependencies:

`npm install`

Run with:

`npm run start`

The web app is running under port **3000** by default.
